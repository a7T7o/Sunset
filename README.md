# Sunset

> 一款 2D 像素生活模拟 RPG。
> 你在失忆后醒来，被带进一座靠近荒野的聚落。这里没有从天而降的英雄任务，只有一张旧工作台、一片待开垦的土地、一顿并不平静的晚饭，以及那些慢慢回来的记忆。

《Sunset》想做的是一个带剧情重量的生活模拟游戏。
白天，你砍树、采石、整理背包、制作工具、开垦土地；夜晚，你在村庄和自己的过去之间，决定还要不要继续往前走。

当前试玩版开放了 Day1 的主要流程。
你会从被救起开始，进入村落、疗伤、学习使用工作台，并完成第一轮采集、开垦和制作。

## 这是一款什么游戏

玩家扮演一名失忆工匠，在一个勉强维持运转的聚落里重新开始生活。

这个聚落并不是单纯的背景板。它缺少稳定生产，也缺少对外来者的信任。你要先通过劳动证明自己能留下来，再通过制作、探索和一次次闪回，慢慢拼回自己为什么会来到这里。

所以《Sunset》的核心并不是“种地功能集合”，而是把生活劳动、制作成长、人物关系和记忆线放进同一条日常节奏里。

## 当前试玩版能玩到什么

Day1 是当前试玩版的核心内容，它已经能从开场推进到夜晚收束：

1. 在危险边缘被救起，第一次意识到这个世界并不安全。
2. 被带进村庄，在围观、议论和带路中认识这座聚落。
3. 进入旧屋疗伤，看到血量和精力不再只是界面元素，而是剧情里的真实处境。
4. 第一次触摸工作台，触发记忆闪回，重新接通最基础的制作能力。
5. 砍树、采石、锄地、播种、浇水，把第一轮资源变成可以继续生活的东西。
6. 回到村里经历晚饭冲突，看到这个聚落并不会轻易接纳你。
7. 进入夜间自由活动，再决定什么时候回去休息。

这一天的重点不是把按钮教完，而是让玩家真的过完“来到这里的第一天”。

## 游戏亮点

- **从第一天开始建立关系**
  被救起、入村、疗伤、劳动、晚饭冲突和夜晚收束不是分散事件，而是一条连续的入村体验。

- **劳动不是菜单，而是剧情的一部分**
  砍树、采石、耕地、播种和制作并不是单独摆出来的功能，它们共同回答一个问题：你能不能靠自己的手在这里留下来。

- **工作台连接制作和记忆**
  工作台不仅是制作入口，也是主角记忆闪回的触发点。玩家第一次接触制作时，同时也在接触自己的过去。

- **生活系统被接进同一条流程**
  背包、快捷栏、箱子、工作台、农田、采集、时间推进、存档读档和跨场景切换，已经在 Day1 里服务同一段体验，而不是各自孤立展示。

- **村庄不是安全背景板**
  NPC 的围观、带路、日常承接和晚饭冲突，会让玩家感到这里有人、有秩序，也有不欢迎你的立场。

## 当前已经落地的内容

当前版本主要围绕 Day1 试玩切片展开，已接入并打通以下内容：

- `Town`、`Primary`、`Home` 三个场景之间的主线承接
- 开场救起、入村围观、疗伤、工作台闪回、农田教学、晚饭冲突、夜间自由活动等剧情节点
- 树木、石头、工具、作物、农田、工作台和基础制作
- 36 格背包、12 格快捷栏、箱子存储与物品交互
- 血量、精力、时间推进、夜晚收束与基础状态承接
- 当前试玩范围内的存档、读档和跨场景继续
- NPC 的围观、带路、散场和日常承接基础

这些内容的目标不是单纯证明“系统数量多”，而是让 Day1 可以从剧情进入劳动，再从劳动回到人物关系和夜晚节奏。

## 仍在扩展的方向

下面是项目已经确定的后续开发方向，不代表当前试玩版已经全部开放：

- 聚落会从“暂时能活”继续走向“逐步恢复功能”，后续会扩展设施、秩序、居民日常和复兴反馈。
- 工作台会继续往分站点生产链扩展，例如熔炉、铁砧、烹饪等方向，让制作不只停留在 Day1 的基础阶段。
- 钥匙与上锁宝箱会继续作为探索中的风险收益设计，和背包空间、资源投入、矿洞威胁形成取舍。
- 四季、材料等级、资源门槛和工具成长会继续拉长经营节奏，让同一张地图在不同阶段有不同劳动重点。
- 记忆闪回、矿洞威胁和人物关系会继续推进主线，让“找回自己”和“复兴聚落”互相牵引。
