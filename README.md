# Sunset

> 醒来时，你已经欠下一条命。

《Sunset》是一款剧情驱动的 2D 像素生活模拟 RPG，也是我的个人游戏 Demo。

你扮演一名失忆工匠，被荒野边缘的聚落救回。这里给了你一间旧屋、一张旧工作台、一片还没开垦的土地，也给了你怀疑、饭桌上的冲突，以及那些慢慢回来的记忆。

在《Sunset》里，种地不是单纯填格子，砍树也不是为了刷材料。白天，你用斧头、锄头、背包和工作台争取留下来的资格；夜晚，你在村庄和自己的过去之间，判断自己还要不要继续往前走。

当前公开试玩版开放 Day1 完整路线：被救起、入村、疗伤、工作台闪回、采集开垦、制作整理、晚饭冲突和夜间自由活动。

## 你会经历什么

Day1 不是一段把按钮教完的新手教程，而是主角来到聚落后的第一天。

你会先在危险边缘被救起，被带进一座并不完全欢迎你的村庄。村民会围观、议论、带路，也会在晚饭时把怀疑摆到桌面上。你不是一进门就被接纳的英雄，而是一个必须先证明自己能留下来的外来者。

这一天里，你会进入旧屋疗伤，第一次接触工作台，在闪回里重新接上制作能力；随后出门砍树、采石、整理背包、开垦土地、播种浇水，把手里的资源变成第二天还能继续生活的基础。

到了夜里，村庄安静下来。玩家可以自由活动、整理物品、确认状态，再选择什么时候回去休息。Day1 的目标不是炫系统数量，而是让玩家真的过完“被救之后的第一天”。

## 游戏亮点

### 第一天不是教程，是入村考验

《Sunset》把开场做成一条连续的生活线：被救、入村、疗伤、劳动、晚饭冲突、夜晚收束。玩家不是被菜单推着走，而是在一连串具体事件里认识这个聚落。

### 劳动不是菜单，是留下来的理由

砍树、采石、锄地、播种、浇水、制作，这些行为不是孤立功能。它们共同指向一个更朴素的问题：你能不能靠自己的手，在这个地方活下来。

### 工作台既做工具，也撬开记忆

工作台不是单纯的合成入口。主角第一次触碰它时，制作能力和记忆闪回会一起发生。它既是生产工具，也是主线叙事的一扇门。

### 村庄救了你，也在审视你

这个聚落不是安全背景板。NPC 的围观、带路、散场、日常承接和晚饭冲突，会让玩家感到这里有人、有秩序，也有不欢迎你的立场。

### 系统被压进同一天，而不是摊成展示柜

背包、快捷栏、箱子、工作台、农田、采集、时间推进、存档读档和跨场景切换，都已经接进 Day1 的体验里。它们不是摆出来证明“我做了很多系统”，而是一起服务主角从醒来到入夜的第一天。

## 后续开发方向

下面是《Sunset》已经确定的设计方向，不代表当前试玩版已经全部开放。

- 聚落会从“勉强能活”逐步走向“恢复功能”，后续会扩展设施、居民日常、秩序变化和复兴反馈。
- 工作台会继续发展成生产链入口，向熔炉、铁砧、烹饪等分站点扩展，让制作从 Day1 的基础生存走向长期经营。
- 钥匙、上锁宝箱和矿洞威胁会进入探索结构，和背包空间、资源投入、风险收益形成取舍。
- 四季、材料等级、工具成长和资源门槛会拉长经营节奏，让同一张地图在不同阶段有不同劳动重点。
- 记忆闪回、人物关系和聚落复兴会继续互相牵引，让“找回自己”和“让这里重新运转”成为同一条主线。
