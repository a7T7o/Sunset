# Sunset

> 一款仍在开发中的 2D 像素奇幻生活模拟 RPG。
> 当前重点是先把主角来到这里的第一天做扎实。

《Sunset》的主角是一名失忆工匠。
他在危险边缘被人救起，被带进一个勉强维持运转的聚落，先学会怎么活下来，再慢慢弄明白自己为什么会来到这里。

如果你第一次打开这个仓库，最建议先看这两个入口：

- [完整演示视频](https://www.bilibili.com/video/BV16Ed1BAEun/)
- [Windows 试玩包（Release）](https://github.com/a7T7o/Sunset/releases/latest)

## 当前版本是什么状态

当前仓库对应的是一个已经能玩通 Day1 主线切片、但仍在继续打磨中的版本。
它已经能完整走完第一天，但离长期内容齐备的正式成品还很远。

现在最稳定、也最值得看的，是这条已经能从头推进到夜里的 Day1：

- 被救起
- 被带进村
- 被安置和疗伤
- 第一次摸到工作台并触发记忆闪回
- 完成第一轮开垦、播种、浇水、砍树和基础制作
- 经历晚饭上的冲突
- 进入夜间自由活动，再决定什么时候回去休息

这条线已经能把剧情、引导、劳动、制作和时间节奏串到一起，
目前仍在继续修整导演尾差、玩家提示和部分运行时细节。

## 现在实际能玩到什么

如果只从玩家能直接感受到的东西来讲，当前版本已经有这些内容：

- 村庄、采集区和旧屋之间的 Day1 场景承接
- 入村围观、带路、疗伤、晚饭冲突和夜间收束这条主线流程
- 血量和精力在剧情里的首次出现，而不是单独摆出来的说明面板
- 背包、快捷栏、箱子和工作台的基础交互
- 砍树、采石、锄地、播种、浇水和第一轮基础制作
- 时间推进、白天到夜晚的节奏变化
- 当前版本的存档、读档和跨场景承接

对这个项目来说，种地、做东西和推进剧情不是分开的三条线。
它们更像是同一件事的不同侧面：你得先靠劳动证明自己能留下来，后面才谈得上更长的生活和更深的记忆线。

## 这个项目想往哪边继续长

下面这些是已经定下来的继续开发方向，但它们不等于“当前版本已经完整实现”：

- 聚落会继续从“暂时能活”往“逐步恢复功能”推进，后面会补更多设施、秩序和居民日常。
- 工作台不会只停在 Day1 的基础制作，后续会继续往分站点生产链扩展，例如熔炉、铁砧、烹饪等方向。
- 钥匙与上锁宝箱会继续作为探索中的风险收益玩法存在，而不是只做成普通容器。
- 四季、材料等级和资源门槛会继续把经营线拉长，让同一张地图在不同阶段有不同的劳动重点。
- 记忆闪回、矿洞威胁和人物关系不会停在开场，它们会继续推动后面的剧情展开。

## 这个仓库里主要有什么

这个仓库现在主要放三类东西：

- 项目源码
- 持续更新中的设计与阶段文档
- 当前版本的试玩入口与说明

如果你是想直接体验，优先看视频或下载试玩包。
如果你是想继续研究项目结构，再往下看文档入口会更合适。

## 如何打开项目

- 推荐 Unity 版本：`6000.0.62f1`
- 直接以仓库根目录作为 Unity 项目打开
- 当前主线切片围绕 `Town`、`Primary` 和 `Home` 展开
