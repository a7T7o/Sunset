- 新的一轮 prompt，如果会产出独立一批文档，就新建下一个批次文件夹。
- 每个批次文件夹里至少放：
  - 本轮 prompt
  - 本轮直接产出的文档
- 根目录尽量只保留：
  - `memory_0.md`
  - `README.md`
