# 题目3：滑动窗口协议仿真 (C#版本)

## 功能说明
模拟选择性重传(Selective Repeat)滑动窗口协议的工作过程。

## 编译方法
```bash
csc SlidingWindowSimulator.cs -out:SlidingWindow.exe
```

## 技术要点
1. **选择性重传协议**：只重传丢失的帧，不影响其他帧
2. **窗口大小限制**：使用n位序号时，窗口最大为2^(n-1)
3. **超时重传**：帧超时未确认则重传
4. **乱序接收**：接收方可以乱序接收帧，按序交付

## 协议参数
- 序号范围：0-7（3位）
- 窗口大小：4
- 超时时间：3000ms
- 模拟丢包率：20%

## 显示内容
- 发送窗口状态
- 接收窗口状态
- 帧发送/接收/确认/重传事件
- 统计信息（发送数、确认数、丢失数、重传数、效率）
