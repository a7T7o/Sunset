# 题目5：网络路由模拟器 (C#版本)

## 功能说明
模拟网络路由器的数据包转发过程。

## 编译方法
```bash
csc NetworkRouterSimulator.cs -out:RouterSim.exe
```

## 技术要点
1. **路由表**：存储目标网络、子网掩码、下一跳、出接口
2. **最长前缀匹配**：选择最精确的路由条目
3. **数据包转发**：根据路由表逐跳转发
4. **TTL处理**：防止数据包无限循环

## 网络拓扑
```
   [PC1]                                           [PC4]
 192.168.1.10                                   192.168.4.10
      |                                              |
   [R1]----[R2]----[R3]----[R4]
      |                      |
   [PC2]                  [PC3]
 192.168.1.20          192.168.3.10
```

## 网段划分
- 192.168.1.0/24: R1的LAN
- 10.0.12.0/30: R1-R2链路
- 10.0.23.0/30: R2-R3链路
- 10.0.34.0/30: R3-R4链路
- 192.168.3.0/24: R3的LAN
- 192.168.4.0/24: R4的LAN
