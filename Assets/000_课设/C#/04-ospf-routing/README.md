# 题目4：OSPF路由协议原型系统 (C#版本)

## 功能说明
模拟OSPF链路状态路由协议的工作过程。

## 编译方法
```bash
csc OSPFSimulator.cs -out:OSPF.exe
```

## 技术要点
1. **链路状态包(LSP)**：每个节点生成包含邻居信息的LSP
2. **泛洪机制**：LSP在网络中泛洪传播
3. **链路状态数据库(LSDB)**：每个节点维护完整的网络拓扑
4. **Dijkstra算法**：计算到所有节点的最短路径
5. **路由表生成**：根据最短路径生成路由表

## 网络拓扑
```
        2       1
    A ───── B ───── C
    │       │       │
   5│      2│      8│
    │       │       │
    D ───── E ───── F
        1       4
```

## 仿真步骤
1. 各节点生成LSP
2. LSP泛洪传播
3. 使用Dijkstra算法计算路由
4. 生成路由表
5. 模拟数据包转发
