# C#/Unity 版本 - 挂载到空物体运行

## 使用方法
1. 把对应的.cs文件复制到Unity项目的Assets文件夹
2. 创建空物体 (Ctrl+Shift+N)
3. 把脚本拖到空物体上
4. 点Play运行

## 实验1: 聊天程序
位置: `01-chat-program/`
- `ChatServer.cs` - 服务器 (先运行)
- `ChatClient.cs` - 客户端

## 实验3: 滑动窗口
位置: `03-sliding-window/`
- `SlidingWindow.cs` - 点按钮模拟发送

## 实验4: OSPF路由
位置: `04-ospf-routing/`
- `OSPF.cs` - 点节点按钮查看路由表

## 实验5: 路由模拟器
位置: `05-network-router-simulator/`
- `RouterSim.cs` - 模拟数据包转发
