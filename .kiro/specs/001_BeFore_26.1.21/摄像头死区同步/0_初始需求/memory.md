# 摄像头死区同步 - 开发记忆

## 模块概述

实现 Cinemachine 摄像头死区（Dead Zone）与场景边界的自动同步功能。当场景加载或切换时，系统自动检测场景边界并计算死区参数，避免摄像头显示场景外的空白区域。

## 核心功能

- 场景加载完成后自动检测场景边界
- 场景切换时重新计算并应用死区参数
- 玩家移动到场景边缘时摄像头停止跟随
- 自动获取场景大小，无需手动配置
