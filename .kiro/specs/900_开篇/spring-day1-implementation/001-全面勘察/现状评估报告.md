## 二、关键系统深度分析

### 2.1 时间系统 (TimeManager)

**核心机制**:
```csharp
// 时间流逝配置
- 1游戏天 = 1200秒现实时间（20分钟）
- 游戏时间: 06:00 - 02:00（20小时）
- 每小时6次分钟跳跃（每10分钟）
- 每季28天，4季循环

// 关键方法
public void Sleep()  // 睡觉 → 触发OnSleep事件 → 进入下一天
public void AdvanceDay()  // 日期推进 → 触发OnDayChanged
```

**事件系统**:
- `OnMinuteChanged` - 分钟变化（UI更新）
- `OnHourChanged` - 小时变化（光照、NPC日程）
- `OnDayChanged` - 每日变化（作物生长、树木成长）
- `OnSeasonChanged` - 季节变化（视觉切换）
- `OnYearChanged` - 年份变化
- `OnSleep` - 睡觉事件（**关键：开篇需要监听此事件**）

**开篇相关**:
- ✅ 已有睡眠触发机制
- ✅ 已有日期推进逻辑
- ⚠️ 缺少开篇剧情触发器（需要在首次Sleep时检测）

**开篇相关**:

## 四、开篇实现方案建议

### 4.1 最小化实现方案（推荐）

**核心思路**: 利用现有系统，最小化新增代码

#### 步骤1: 创建开篇触发器
```csharp
// 新建: OpeningTrigger.cs
public class OpeningTrigger : MonoBehaviour
{
    void Start()
    {
        // 监听首次睡眠事件
        TimeManager.OnSleep += CheckFirstSleep;
    }

    void CheckFirstSleep()
    {
        if (!PlayerPrefs.HasKey("OpeningCompleted"))
        {
            TriggerOpening();
            PlayerPrefs.SetInt("OpeningCompleted", 1);
        }
    }

    void TriggerOpening()
    {
        // 显示开篇文本
        ShowOpeningText();
    }
}
```

#### 步骤2: 创建简易文本显示器
```csharp
// 新建: SimpleTextDisplay.cs
// 使用Unity UI Text显示开篇文本
// 无需复杂对话系统
```

#### 步骤3: 修改时间系统
```csharp
// 修改: TimeManager.cs
// 在Sleep()方法中添加首日检测
public void Sleep()
{
    OnSleep?.Invoke();  // ← 已有，无需修改
    AdvanceDay();

    // 新增: 首日特殊处理
    if (currentYear == 1 && currentDay == 1)
    {
        OnFirstDayWakeUp?.Invoke();
    }
}
```
