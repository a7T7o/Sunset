### 1. 剧情数据结构（第238-239行）

**修正前**：
```
- StoryStep:
  - StepType type (Dialogue/CG/GiveItem/MoveNPC)
  - string data (JSON格式存储具体数据)
```

**修正后**：
```
- List<StoryAction> actions - 使用 [SerializeReference] 多态序列化
- StoryAction (抽象基类):
  - abstract void Execute()
- 具体动作子类：
  - PlayDialogueAction : StoryAction
  - ShowCGAction : StoryAction
  - GiveItemAction : StoryAction
  - MoveNPCAction : StoryAction
```

**修正原因**：
- 原方案使用 `enum + JSON字符串` 是反模式
- Unity 2020+ 支持 `[SerializeReference]` 多态序列化
- 新方案类型安全、易扩展、Inspector友好
