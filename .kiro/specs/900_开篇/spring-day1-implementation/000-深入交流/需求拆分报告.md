## 🛠️ 基础设施清单

### 1. 对话系统（R1）

**现状**: ❌ 无

**最简实现方案**:
```
组件结构：
- DialogueBox (Canvas UI)
  - Background (Image) - 黄色占位矩形
  - NameText (TextMeshPro) - 显示说话者名字
  - ContentText (TextMeshPro) - 显示对话内容
  - ContinueButton (Button) - "继续"按钮

核心脚本：
- DialogueManager.cs - 单例，管理对话流程
  - ShowDialogue(string speaker, string content)
  - HideDialogue()
  - IsDialogueActive() - 供其他系统查询

数据结构：
- DialogueData.cs - ScriptableObject
  - List<DialogueLine> lines
  - DialogueLine { string speaker, string content }
```

**你的困惑点**:
> "对话框如何根据文本大小自适应？"

**回答**: 
- 第一阶段：**不做自适应**，固定高度，文本超出就滚动
- 后续优化：使用 `ContentSizeFitter` 组件自动调整高度

**扩展方向**:
- 打字机效果（逐字显示）
- 头像显示
- 选项分支
- 语音播放

### 2. 事件触发系统（R4）

**现状**: ❌ 无

**最简实现方案**:
```
组件结构：
- EventTrigger (GameObject + BoxCollider2D)
  - 挂载 EventTriggerZone.cs
  - 配置触发条件（进入区域、按E键、持有特定物品）

核心脚本：
- EventTriggerZone.cs - 区域触发器
  - OnTriggerEnter2D() - 玩家进入
  - CheckCondition() - 检查触发条件
  - TriggerEvent() - 触发事件

- EventManager.cs - 单例，事件注册中心
  - RegisterEvent(string eventId, UnityEvent callback)
  - TriggerEvent(string eventId)
  - IsEventCompleted(string eventId) - 检查事件是否已完成

数据结构：
- EventData.cs - ScriptableObject
  - string eventId - 事件唯一ID
  - EventType type - 类型（对话、CG、物品获得）
  - List<EventCondition> conditions - 触发条件
```

**你的困惑点**:
> "是否需要顶层管理器？还是事件注册机？"

**回答**:
- 第一阶段：**简单的事件注册机**（EventManager单例）
- 每个事件有唯一ID（如"spring_day1_wake_up"）
- 事件触发后调用 `EventManager.TriggerEvent(id)`
- 其他系统订阅事件（如对话系统订阅"spring_day1_wake_up"显示对话）

**扩展方向**:
- 事件依赖链（事件A完成后才能触发事件B）
- 事件状态持久化（存档）
- 事件回放系统

### 6. NPC引导系统（R6）

**现状**: ❌ 无

**最简实现方案**:
```
核心脚本：
- NPCFollowPlayer.cs - 挂载到NPC上
  - float stopDistance = 3f - 停止距离
  - float slowDistance = 5f - 减速距离
  - float matchDistance = 2f - 匹配速度距离
  - Update() - 根据距离调整速度
```

**你的理解**:
> "npc先走，走到一个阈值就停，此时玩家再走npc就先慢慢走，然后再进入一个阈值npc就开始匹配玩家速度..."

**回答**:
- 完全正确！实现逻辑：
```csharp
float distance = Vector2.Distance(npc.position, player.position);
if (distance > slowDistance) {
    npc.speed = normalSpeed; // 正常速度前进
} else if (distance > matchDistance) {
    npc.speed = slowSpeed; // 慢速前进
} else if (distance > stopDistance) {
    npc.speed = player.speed; // 匹配玩家速度
} else {
    npc.speed = 0; // 停止
}
```

**扩展方向**:
- 路径点导航（NPC沿预设路径移动）
- 避障逻辑
- 动画状态同步
