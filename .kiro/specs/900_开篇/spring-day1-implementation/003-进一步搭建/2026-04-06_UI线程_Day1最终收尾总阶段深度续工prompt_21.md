这是你当前最核心的主战场，不要再漂。

本轮必须尽全力推进：

1. 左列 recipe generated row 真显示

2. 右侧详情区真可读
3. 制作条 / 制作按钮 / 领取 / 取消 / 追加 的玩家语义真一致
4. 悬浮说明、进度提示、小框布局真可判
5. 面板与工作台的相对稳定表现
6. 上下切换、hover、click 的玩家体感
