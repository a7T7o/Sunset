# spring-day1 非 UI 剧情扩充框架落地任务单

## 1. 目标

这份任务单只服务一件事：

在 **不碰 UI 主刀内容、不回推现有已跑通链** 的前提下，把春一日最初原案缺失的剧情桥段，用 `spring-day1` 自己能继续拥有的逻辑 / 剧情 / NPC 出场框架补齐。

本任务单默认只管：

1. 阶段语义
2. 剧情资产
3. 非 UI 触发与承载
4. NPC 出场时机
5. temp scene -> Town 的交接边界

---

## 2. 框架总原则

## 2.1 不重砍 `StoryPhase`

当前不建议第一刀就改 `StoryPhase` 枚举。

原因：

1. 现有测试、导演逻辑、提示链都已经围绕现有 9 个 phase 跑通。
2. 当前最缺的是“每个 phase 内部太薄”，不是“phase 名字完全错了”。
3. 粗暴改 phase 会把现在已经稳定的逻辑一起打散。

因此后续推荐口径是：

1. **对外仍保留现有 9 个 phase。**
2. **在 phase 内增加更细的剧情步 / beat。**

## 2.2 先改剧情资产，再改表现层

后续应先把“谁说什么、谁什么时候出现”落成逻辑可控资产，再交给 UI 线程接显示。

优先级固定为：

1. 剧情语义
2. 事件触发
3. NPC 出场
4. UI 显示

## 2.3 先守正式角色，再用 crowd 补背景

后续实现的角色优先级固定为：

1. 马库斯
2. 艾拉
3. 卡尔
4. 老杰克 / 老乔治 / 老汤姆 / 小米
5. 围观村民 / 饭馆村民 / 小孩
6. `101~301` 只作为 crowd 壳或过桥槽位

## 3. 分阶段扩充任务

## A. `CrashAndMeet` 内部扩充

### A1. 补成 4 个剧情步

`CrashAndMeet` 后续应拆成：

1. `WakeAtMine`
2. `LanguageDecode`
3. `MonsterCue`
4. `FollowMarcusEscape`

### A2. 这一刀要补的内容

1. 把“首段正式对话”前后增加危险感桥段。
2. 增加一个明确的怪物逼近触发。
3. 让跟随 / 奔跑教学发生在这里，而不是凭空开始。

### A3. 这一刀不碰什么

1. 不改 UI 教学卡排版
2. 不改输入系统总入口
3. 不在这里引入 Town 群像大规模出现

---

## B. `EnterVillage` 内部扩充

### B1. 补成 4 个剧情步

`EnterVillage` 后续应拆成：

1. `VillageGateArrival`
2. `CrowdGaze`
3. `AbandonedHouseArrival`
4. `HandOffToElla`

### B2. 这一刀要补的内容

1. 进村时至少要有围观感。
2. 明确这是闲置小屋，不是大儿子的房子。
3. 让马库斯把玩家正式交给艾拉，而不是直接切进疗伤。

### B3. 角色约束

1. 这里能说话的核心仍以马库斯为主。
2. 村民可以短句或无名反应，但不抢艾拉出场。
3. 小米 / 小孩可以只作为视线存在，不必马上展开对话。

## 6. 和 UI 线程的接口约定

后续和 UI 对接时，这条线只负责给出：

1. 当前阶段到底是什么
2. 当前桥段焦点是谁
3. 当前任务是否推进
4. 当前夜间压力是什么
5. 当前谁可交互、谁只做背景

UI 线程负责：

1. 怎么显示任务
2. 怎么显示提示
3. 怎么显示卡片和动画
4. 怎么把这些结果呈现得顺眼

也就是说：

1. **剧情语义在这里。**
2. **玩家看到的形状在 UI 那边。**

---

## 7. 推荐施工顺序

后续如果继续真实落地，建议按这个顺序切：

1. 补 `CrashAndMeet / EnterVillage` 内部剧情步
2. 补 `HealingAndHP / WorkbenchFlashback` 的正式剧情资产
3. 补 `DinnerConflict / ReturnAndReminder` 的剧情资产化
4. 补 `FreeTime` 的 Town 夜间见闻包
5. 最后再做 `Town` 正式迁移

这个顺序的原因很简单：

1. 先把 Day1 前半段回正，玩家才会觉得“故事真的开始了”。
2. 再把中段教学和晚间收束接稳，Day1 才会像完整的一天。
3. 等这些都稳了，再把大部分 NPC 迁回 `Town`，不会反过来被场景依赖卡死。

---

## 8. 本轮可直接沿用的执行口径

如果后续继续施工，这条线当前的唯一正确口径是：

1. 不碰 UI 主刀内容
2. 不推翻现有功能链
3. 不把 `101~301` 当原案正式角色真名
4. 先补原剧本前半段和村庄承接
5. 再补晚间收束和 Town 见闻
