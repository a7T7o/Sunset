## 2. 我这轮要做的具体需求

### 2.1 `HealingAndHP`

要补成正式剧情版，而不是只剩功能开关。

至少补清：

1. 疗伤这段的情绪承接
2. HP 显现和剧情语义之间的关系
3. 艾拉接手疗伤时的角色功能
4. 疗伤结束后为什么要接到工作台闪回

### 2.2 `WorkbenchFlashback`

现在不能只剩“打开工作台 -> 过段”。

至少补清：

1. 为什么是工作台
2. 这段在 Day1 故事里的意义
3. 它怎么把玩家从“被救”推进到“开始在村里活下去”

### 2.3 `FarmingTutorial`

这段不能只是一串功能教学。

要补成：

1. 开垦
2. 播种
3. 浇水
4. 收木
5. 制作

背后都带剧情语义：

- 不是学系统
- 是开始真正安顿下来

### 2.4 `DinnerConflict`

这段现在还是薄。

要补清：

1. 晚饭这段的冲突中心
2. 角色立场
3. 玩家为什么会被卷进去
4. 它为什么要把 Day1 带进夜晚提醒

### 2.5 `ReturnAndReminder -> FreeTime -> DayEnd`

这段要补成有结束感的正式尾段。

至少补清：

1. 夜晚提醒怎么递进
2. 自由活动为什么不是“啥都没有”
3. 回屋/睡觉为什么构成 Day1 收束
4. DayEnd 的结束感和第二天前的余味

### 2.6 全阶段桥接矩阵总清盘

这刀里必须一起补的还有：

1. 每段对白结束后推到哪个 `StoryPhase`
2. 哪些 UI/系统状态此时应显/应隐
3. 哪些阶段必须 formal 优先
4. 哪些异常路径要兜底
5. 时间、血量、精力、任务、对话之间有没有桥接漏口
