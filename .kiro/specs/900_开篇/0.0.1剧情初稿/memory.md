## 关键设计决策

### 第一天剧情流程（确定版）

**开场CG**（约30秒）：
- 黑屏→星空→流星（主角）坠落→砸进矿洞口附近的森林→失去意识

**14:00 坠落与相遇**：
- 主角在矿洞口外醒来
- 村长马库斯出现，说话显示乱码
- 主角回忆起古语，乱码变正常中文
- 远处矿洞口有骷髅兵徘徊（未发现玩家）
- 村长带主角离开，教学移动和奔跑

**14:20 初入村庄**：
- 村民好奇围观（主角衣着体型与村民不同）
- 村长带主角到废弃小屋

**14:30 疗伤与血条UI**：
- 艾拉（村长女儿）治疗主角
- 血条UI出现（此时精力条尚未出现）

**14:45 工作台与记忆闪回**：
- 主角触摸工作台，记忆闪回
- 解锁手工制作配方
- 艾拉被主角的"胡话"吓到

**15:10 耕种教学与精力条UI**：
- 村长教主角种田（花菜种子）
- 开垦第一格后，精力条UI出现
- 完成3格种植（开垦+播种+浇水）

**16:00 晚餐与冲突**：
- 村长带主角去饭馆
- 村长二儿子卡尔挑衅
- 吃饭恢复精力

**17:00 归途与夜间提醒**：
- 村长提醒"晚上两点前必须睡觉"
- 主角回到小屋
- 可自由活动或睡觉
