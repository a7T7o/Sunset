# 春1日 - 坠落 | 格式B：分镜脚本版

## 场景1：开场CG（约30秒）

### 分镜1-1
- **场景描述**：黑屏后，星空逐渐显现。
- **镜头**：全黑 → 慢淡入到广角星空（固定镜头）。
- **UI提示**：无UI。
- **代码实现提示**：`PlayableDirector` 驱动 Timeline；使用 `CanvasGroup` 做黑场淡入。

### 分镜1-2
- **场景描述**：一颗流星划过天际（主角）。
- **镜头**：横向跟拍流星，末段快速推近坠落点。
- **UI提示**：无UI。
- **代码实现提示**：流星可用 VFX Graph/粒子；落点触发屏幕震动（Cinemachine Impulse）。

### 分镜1-3
- **场景描述**：主角坠入矿洞口附近森林，昏迷。
- **镜头**：俯拍坠落坑 → 缓慢拉近主角面部 → 渐黑。
- **UI提示**：无UI。
- **代码实现提示**：落地时播放一次性 SFX；切到可控状态前锁输入。

## 场景2：14:00 坠落与相遇（矿洞口外）

### 分镜2-1
- **场景描述**：主角醒来，视野模糊，村长马库斯靠近。
- **镜头**：主角主观镜头（轻微模糊）→ 切第三人称中景。
- **UI提示**：无UI。
- **代码实现提示**：后处理景深/模糊短时开启；对话前禁用玩家移动。

### 分镜2-2（语言乱码转换）
- **场景描述**：马库斯初始发言为乱码，主角触发古语回忆后恢复正常中文。
- **镜头**：马库斯近景说话 → 插入主角记忆闪回（快速闪帧）→ 回到马库斯中景。
- **UI提示**：对话框文本从乱码切换为正常中文。
- **代码实现提示**：
  - 对话节点A：`garbledText`。
  - 回忆触发后置 `languageDecoded=true`。
  - 后续节点走 `normalText` 分支，单次切换不回退。

### 分镜2-3（骷髅兵威胁）
- **场景描述**：矿洞口远处有骷髅兵徘徊，但未发现玩家与村长。
- **镜头**：长焦远景扫到矿洞口，骷髅兵剪影移动。
- **UI提示**：无战斗UI。
- **代码实现提示**：骷髅兵处于巡逻状态，感知半径外；仅做“危险提示”不触发战斗。

### 分镜2-4（移动教学）
- **场景描述**：马库斯示意立刻离开危险区域。
- **镜头**：跟随镜头切为可操作状态。
- **UI提示**：
  - `使用 WASD 移动`
  - `按住 Shift 奔跑`
- **代码实现提示**：教学 Trigger 分阶段校验输入（移动成功/奔跑成功）后解锁下一段。
