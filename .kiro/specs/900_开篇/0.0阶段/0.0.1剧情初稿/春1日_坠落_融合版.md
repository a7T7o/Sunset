## 开场CG（约20秒）

> 实现方案：静态图 + 文字叙述 + 音效（A+C 融合方案）

### CG-1：星空
- **场景描述**：黑屏后，星空静态图逐渐显现。
- **画面**：静态插画（星空夜景）。
- **文字**（底部淡入）："那一夜，星空格外宁静..."
- **音效**：寂静 → 微弱虫鸣。
- **停留**：3 秒。
- **代码提示**：`Image` 组件 + `CanvasGroup` 控制淡入；`TextMeshPro` 文字淡入；Timeline 驱动序列。

### CG-2：流星坠落
- **场景描述**：流星划过天际的静态图（或叠加简单粒子拖尾）。
- **画面**：静态插画（流星划过星空）或在 CG-1 基础上叠加粒子特效。
- **文字**："直到一颗流星划破天际。"
- **音效**：流星破空声。
- **停留**：2 秒。
- **代码提示**：可选用 `Trail Renderer` 做简单拖尾；音效同步播放。

### CG-3：坠落
- **场景描述**：森林俯视图，中央有坠落形成的坑。
- **画面**：静态插画（森林 + 坠落坑）。
- **文字**："它坠入了这片被遗忘的土地。"
- **音效**：沉闷撞击声 + 屏幕轻微震动。
- **停留**：2 秒。
- **代码提示**：`Cinemachine Impulse` 做屏幕震动（可选）。

### CG-4：昏迷
- **场景描述**：画面渐黑，主角内心独白。
- **画面**：渐黑。
- **文字**："而我...就是那颗流星。"
- **音效**：心跳声渐弱 → 寂静。
- **停留**：2 秒。
- **代码提示**：`CanvasGroup.alpha` 渐变至 0；播放完毕后切换到游戏场景，锁定玩家输入直到村长出现。

## 14:00 - 坠落与相遇

**地点**：矿洞口外森林

### 2-1：醒来
- **场景描述**：主角醒来，视野模糊，村长马库斯靠近。
- **镜头**：主角主观镜头（轻微模糊）→ 切第三人称中景。
- **UI**：无。
- **代码提示**：后处理景深/模糊短时开启；对话前禁用玩家移动。

### 2-2：语言乱码转换
- **场景描述**：马库斯初始发言为乱码，主角触发古语回忆后恢复正常中文。
- **镜头**：马库斯近景说话 → 插入主角记忆闪回（快速闪帧）→ 回到马库斯中景。
- **UI**：对话框文本从乱码切换为正常中文。
- **代码提示**：对话节点A走 `garbledText`；回忆触发后置 `languageDecoded=true`；后续走 `normalText` 分支，单次切换不回退。

**对话**：
- **马库斯**（乱码）："◆■▲●？！"
- **主角**（内心）："这是...什么语言？等等，这些音节...我好像在哪里听过..."
- **主角**（记忆闪回）：古代语言学课程的片段闪过
- **马库斯**（正常）："年轻人，你还好吗？从天上掉下来，居然还活着..."
- **主角**："我...我在哪里？"
- **马库斯**："这里是落日村。你看起来不像本地人...算了，先离开这里吧。"
- **主角**："为什么？"
- **马库斯**（指向矿洞口）："那边有怪物。虽然它们现在没发现我们，但不能久留。"

### 2-3：骷髅兵威胁
- **场景描述**：矿洞口远处有骷髅兵徘徊，未发现玩家与村长。
- **镜头**：长焦远景扫到矿洞口，骷髅兵剪影移动。
- **UI**：无战斗UI。
- **代码提示**：骷髅兵处于巡逻状态，感知半径外；仅做危险提示不触发战斗。

### 2-4：移动教学
- **场景描述**：马库斯示意立刻离开危险区域。
- **镜头**：跟随镜头切为可操作状态。
- **UI**：`使用 WASD 移动` → `按住 Shift 奔跑`
- **代码提示**：教学 Trigger 分阶段校验输入（移动成功/奔跑成功）后解锁下一段。

**引导操作**：马库斯带路，主角跟随前往村庄。

## 15:10 - 耕种教学与精力条UI

**地点**：废弃小屋外的农田

### 6-1：教学开始
- **场景描述**：马库斯发放锄头与花菜种子，开始耕地教学。
- **镜头**：农田斜俯视教学镜头。
- **UI**：分步提示 → `使用锄头开垦土地` → `选择种子播种` → `使用水壶浇水`
- **代码提示**：教学分三步任务链：开垦→播种→浇水。

**对话**：
- **马库斯**："既然你暂时没地方去，就先住在这里吧。不过，你得自己养活自己。"
- **主角**："我...我会种田吗？"
- **马库斯**："很简单。拿着这把锄头，先开垦土地。"
- **马库斯**："然后播种，最后浇水。记住，作物需要每天浇水。"

### 6-2：精力条首次出现（🔴 关键UI节点）
- **场景描述**：主角开垦第一格后感到疲惫，精力条出现。
- **镜头**：锄头落地动作特写 → UI区域推焦。
- **UI**：**精力条首次出现**（蓝色，血条下方），初始值 `80/200`（坠落受伤，精力大幅削减），开垦后变为 `78/200`。
- **代码提示**：第一次成功开垦后触发 `staminaUIVisible=true`。

**对话**：
- **主角**（开垦第一格后）："好累...我的体力..."
- **马库斯**："你的精力不多了。晚上好好休息，明天就会恢复。"

### 6-3：八格完整种植
- **场景描述**：完成8格"开垦+播种+浇水"（一袋种子8颗）。
- **镜头**：连续动作剪辑（前3格详细，后5格快剪）。
- **UI**：精力实时变化，耕地 -2/格，浇水 -2/格，播种不耗精力，总消耗 `-32`，结束显示 `48/200`。
- **代码提示**：精力消耗由 SOData 配置，UI 实时更新。

**对话**：
- **马库斯**（种完后）："不错，这些花菜几天后就能收获了。不过你看起来很累..."
- **主角**（喘气）："确实...感觉快撑不住了。"

### 6-4：砍树教学（🔴 新增）
- **场景描述**：马库斯指向小屋旁的几棵树，教主角砍树获取木材。
- **镜头**：树木近景 → 斧头挥砍动作 → 木材掉落。
- **UI**：`使用斧头砍伐树木` → `拾取木材`
- **代码提示**：砍树消耗 2 精力/次，树木有血量，需多次砍击才能倒下。

**对话**：
- **马库斯**："种田需要工具，工具需要木材。来，我教你砍树。"
- **主角**（砍了几下）："这比种田还累..."
- **马库斯**："所以要合理分配精力。砍几棵就够今天用了。"

**精力消耗**：砍约 15 次（3 棵小树）= -30，当前精力 `48 - 30 = 18/200`

### 6-5：低精力警告（🔴 关键体验节点）
- **场景描述**：精力降至 20 以下，触发低精力警告。
- **镜头**：主角弯腰喘气，画面边缘轻微变暗。
- **UI**：精力条闪烁红色，弹出提示 `精力过低！请尽快进食或休息，否则会晕倒。`
- **代码提示**：精力 < 20 时触发 `LowStaminaWarning` 状态，UI 闪烁 + 移动速度 -20%。

**对话**：
- **马库斯**（担忧）："你脸色很差...走，先去吃点东西。再这样下去你会晕倒的。"

## 关键执行约束

1. 🔴 **UI出现顺序不可错**：血条（14:30治疗时）→ 精力条（15:10耕地时）
2. 🔴 **第一天作物固定**：花菜（能吃饱），不得替换为大蒜（交易用）

5. 🔴 **语言转换一次性完成**：乱码仅出现在首次接触，后续对话正常
6. **冲突伏笔**：卡尔（二儿子）的敌意，为后续剧情埋伏笔
7. **时代对比**：主角现代知识（程序员、工厂）vs 村庄原始技术（草药、手工）
8. **村落技术水平**：全村统一的基础水平，不是"最高级"
9. **时代设定**：玩家从公元1500年穿越到公元500年

## 格式说明（后续剧本模板）

每个分镜节点包含以下字段：
- **场景描述**：发生了什么（叙事层）
- **镜头**：怎么拍（视觉层）
- **UI**：屏幕上显示什么（交互层）
- **代码提示**：怎么实现（技术层）
- **对话**：角色说了什么（剧情层，放在分镜之后或嵌入分镜内）
- **音效**：听到什么（可选，仅关键节点标注）

时间轴作为主线骨架，分镜编号作为场景内细分。
