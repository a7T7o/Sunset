1. 马库斯
2. 艾拉
3. 卡尔

在没有更高权威 exact mapping 之前：

1. 不 claim 这批是正式原案具名角色
2. 不让这批壳压过原案角色表
3. 只把它们用于匿名群众 / 次级背景 / 观察位 / 外壳位

### 3. 夜间见闻不是新支线

`NightWitness_01` 的职责是：

1. 让玩家感到“夜里还有人看见你”
2. 让夜里的规矩变成空气

它不负责：

1. 发布任务
2. 解释世界观
3. 抢走 `FreeTime / DayEnd` 的收束感

### 4. 次日站位不是 Day2 提前开演

`DailyStand_01~03` 现在只承担：

1. 明天这个村子还会继续活着
2. 人会在自己的位置上开始一天

它不承担：

1. 正式 Day2 runtime
2. 精确路线
3. 晨间交互脚本
