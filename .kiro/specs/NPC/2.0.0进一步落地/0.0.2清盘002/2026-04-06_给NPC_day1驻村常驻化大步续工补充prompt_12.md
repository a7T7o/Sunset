## 四、把“正式剧情一次性消耗，之后只走闲聊”写成内容硬约束

你这轮补 resident 语义矩阵时，必须同时守住下面这条：

1. 只要某个 NPC 的正式剧情聊天已经被消费过
2. 后面再次对话时，就不能再回那段正式剧情
3. 后续只能进入：
   - 闲聊
   - resident 日常对话
   - 当前 phase 后的非正式补句
4. 不能让玩家通过重复聊天，把同一段正式剧情重新触发

这意味着你这轮回来的内容，最好明确区分：

1. 哪些是 `one-shot` 正式剧情对话
2. 哪些是 `post-consume` 的日常闲聊池
3. 哪些是 phase 结束后仍可继续存在的 resident 背景句

不要再把它们写成一锅。
