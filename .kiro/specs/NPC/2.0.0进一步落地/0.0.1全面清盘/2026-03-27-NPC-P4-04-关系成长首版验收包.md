## 5. 验收前提 / 入口

- Unity 保持当前打开状态，不需要关闭
- 不需要重开 `Primary.unity`
- 进入菜单：
  - `Tools/NPC/Relationship`
- 当前推荐先用这几个菜单项：
  - `全部清除持久化`
  - `全部设为/陌生`
  - `全部设为/亲近`
  - `打印当前阶段`

## 6. 验收步骤

### 6.1 先把关系阶段清到统一起点

1. 点击：
   - `Tools/NPC/Relationship/全部清除持久化`
2. 再点击：
   - `Tools/NPC/Relationship/全部设为/陌生`
3. 最后点一次：
   - `Tools/NPC/Relationship/打印当前阶段`
4. 看 Console：
   - `001 / 002 / 003` 是否都显示为 `Stranger`

### 6.2 观察同一 NPC 的“陌生”近身反馈

1. 靠近任意一名 NPC，推荐先看：
   - `001`
2. 不要按 `E`
3. 只等近身轻反馈气泡出现
4. 记住这条句子的口气：
   - 是否更克制
   - 是否更像“第一次打照面”

### 6.3 切到“亲近”后，再看同一 NPC

1. 点击：
   - `Tools/NPC/Relationship/全部设为/亲近`
2. 再点一次：
   - `Tools/NPC/Relationship/打印当前阶段`
3. 稍等至少 `12` 秒，再重新靠近刚才同一名 NPC
   - 这是为了避开当前近身反馈的同 NPC 冷却
4. 再看轻反馈气泡：
   - 是否比刚才更熟、更亲、更像已经相处过一段时间

### 6.4 换另一名 NPC spot-check

1. 再看 `002` 或 `003` 任意一人
2. 对比 `陌生` 和 `亲近` 时的差异
3. 看区别是否仍然符合该角色自己的口吻

### 6.5 最后复核持久化

1. 维持当前阶段，不改 scene
2. 再点一次：
   - `Tools/NPC/Relationship/打印当前阶段`
3. 如果你愿意，也可以在当前 Editor 会话里稍后再点一次
4. 看当前值是否还保持你刚刚设定的阶段

## 7. 预期结果

### 7.1 关系阶段模型

- `Stranger / Acquainted / Familiar / Close`
  四档都应已存在
- 菜单切换后，Console 应打印出明确阶段

### 7.2 同一 NPC 的口气变化

- `陌生`
  - 应更克制
  - 更像先保持距离
  - 不会太快进入熟人语气
- `亲近`
  - 应更熟
  - 更像已经建立一定关系
  - 会更自然地关心、留人或延长对话感

### 7.3 角色口吻不能被阶段覆盖掉

- `001`
  - 即使进入 `亲近`
  - 仍应像“稳、照看、顾节奏”
- `002`
  - 即使在 `陌生`
  - 仍应像“轻柔、生活化、愿意多说两句”
- `003`
  - 即使在 `亲近`
  - 仍应保留“观察、推测、记录”的底色

### 7.4 持久化

- 你通过菜单设定的阶段，应能被再次打印出来
- 不应出现：
  - 设成 `亲近` 后一会儿又自己掉回 `陌生`
  - `001` 改了但 `002 / 003` 没跟上菜单的统一设置

## 8. 失败判读

- 情况 A：
  - 四档阶段菜单能点，但气泡内容完全没变化
  - 判读：
    - 关系阶段值变了
    - 内容层分流没真正接上

- 情况 B：
  - 陌生和亲近的句子确实不同
  - 但角色口吻被冲掉了
  - 判读：
    - 分流有了
    - 但内容还不够稳，角色性被阶段性盖过去了

- 情况 C：
  - 菜单设成某阶段后，Console 打印仍不是目标值
  - 判读：
    - 最小持久化接口没有真正写进去

- 情况 D：
  - 切阶段后立刻靠近同一 NPC，但一直看不到新句子
  - 判读：
    - 先优先怀疑当前 `sameNpcRepeatCooldown`
    - 不是先判关系系统失败

- 情况 E：
  - 只有 `001` 会变，`002 / 003` 不变
  - 判读：
    - 某些内容资产的阶段句池没补齐，或映射没接到
