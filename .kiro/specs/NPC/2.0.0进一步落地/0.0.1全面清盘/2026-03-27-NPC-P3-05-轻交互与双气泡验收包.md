  - 把水壶用空后再尝试浇水

## 6. 验收步骤

### 6.1 先看玩家气泡样式

1. 先触发一次玩家自己的短反馈气泡：
   - 推荐优先用“普通斧头砍当前砍不动的树”
   - 或者用“空水壶继续浇水”
2. 只看这一个气泡的样式：
   - 是否明显比 NPC 气泡更浅、更清一点
   - 文字和边框是否仍属于同一套产品语言

### 6.2 再看靠近 NPC 的日常轻反馈

1. 靠近 `001 / 002 / 003` 任意一人，先不要按 `E`
2. 停在近距离一小会儿，观察头顶是否弹出一条短反馈
3. 最好至少看两名 NPC，确认不是所有人都在说同一种口气

### 6.3 再看正式对话轨是否接管成功

1. 在某名 NPC 的日常短反馈刚出现后，直接按 `E` 开正式对话
2. 观察：
   - 刚才那条日常 NPC 气泡是否会及时退掉
   - 正式 `DialogueUI` 是否顺利接管
3. 对话过程中，停留在 NPC 面前，不要离开
4. 看正式对话期间，是否还会继续冒出新的日常 NPC 轻反馈气泡

### 6.4 最后看对话后的恢复节奏

1. 结束正式对话
2. 稍等几秒，再重新靠近或继续停留在 NPC 附近
3. 看日常轻反馈是否会在后续重新出现
4. 同时看它是不是保持了冷却，不会一结束对话就连续刷两三条

## 7. 预期结果

### 7.1 玩家气泡

- 第一眼应更像“自己的念头 / 即时反应”
- 颜色应更清、更浅，不再和 NPC 头顶气泡混成一个观感
- 但又不能像系统提示框或别的游戏 UI

### 7.2 靠近 NPC 的日常轻反馈

- 玩家靠近时，应能看到 NPC 给一条短句
- 这条短句属于“轻响应 / 打招呼 / 近身感觉”
- 不应等正式对话才有任何反馈

### 7.3 正式对话接管

- 按 `E` 进入正式对话后：
  - 原本的日常 NPC 气泡应及时退掉
  - `DialogueUI` 成为唯一主轨
- 正式对话期间：
  - 不应继续冒新的日常近身反馈气泡

### 7.4 对话结束后的恢复

- 正式对话结束后：
  - 日常近身反馈不是永久消失
  - 后续仍应可以再次触发
- 但恢复节奏应是“有冷却的恢复”
  - 不是一结束就立刻连刷
