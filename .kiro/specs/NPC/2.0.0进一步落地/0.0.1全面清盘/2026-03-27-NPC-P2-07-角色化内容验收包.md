  - 判读：

  - 判读：
    - 单人环境表达已分开
    - 玩家轻响应层还没真正完成角色化

- 情况 C：
  - `pairDialogueSets` 为空、partner `npcId` 错、或内容明显还是默认通用句
  - 判读：
    - 相遇矩阵没接实
    - 只能算“角色内容半完成”

- 情况 D：
  - 内容资产写得对
  - 但 `roamProfile` 指到错资产
  - 判读：
    - 这是映射错误，不是文案本身失败

- 情况 E：
  - 静态 Inspector 看起来都对
  - 但 live 里长时间只出现泛化旧句
  - 判读：
    - 优先怀疑运行态取句链仍落回默认句池
    - 不是先去改 scene
