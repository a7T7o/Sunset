   - 放置模式提示在上。
   - 两者同时出现时不打架、不互遮、不跳位失控。

# 用户需求拆分口径

你必须按下面 4 个子问题自己对齐实现：

1. `状态真值`
   - 当前放置模式是否开启，必须直接来自真实运行时状态。
2. `剧情提醒真值`
   - 当前是否该主动提醒玩家开启放置模式，必须直接来自 Day1 director 新补的方法。
3. `表现层`
   - 样式可以沿用交互提示的语言与材料感，但必须明显更像“状态提示”，不能和交互提示完全混淆。
4. `时序层`
   - 切换时：
     - 渐入渐出各 `0.25s`
     - 常显 `2.0s`
     - 总时长约 `2.5s`
   - 再次切换时：
     - 重新触发显示流程
     - 但不要做出多次叠加渐入导致的闪烁 bug
