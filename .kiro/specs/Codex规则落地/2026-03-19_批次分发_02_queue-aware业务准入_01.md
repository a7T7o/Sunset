## 本轮硬规则
1. shared root live 调度命令默认必须走：
   - `powershell -ExecutionPolicy Bypass -File C:\Users\aTo\.codex\tools\sunset-git-safe-sync.ps1`
2. 先 `request-branch`，只有拿到：
   - `GRANTED`
   - 或 `ALREADY_GRANTED`
   才允许继续 `ensure-branch`
3. 如果返回：
   - `LOCKED_PLEASE_YIELD`
   则必须：
   - 写回回收卡
   - 写入自己的 `memory_0.md`
   - 停止继续重试 `ensure-branch`
4. 每条线程本轮只允许做一个最小 checkpoint，做完就：
   - `sync`
   - `return-main`
5. 这轮默认不允许直接进入：
   - `Unity / MCP / Play Mode`
   - `Primary.unity`
   - `GameInputManager.cs`
   除非你的专属 prompt 里明确允许
