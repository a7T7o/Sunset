# 事件编排重构方案 V1

> 本文档是对 Deepseek-2 八篇章剧本的重构方案，核心目标：**压缩时间线、提升事件密度、以配方解锁驱动资源循环**。
> 作者：Claude（基于用户与Kiro确认的方向）
> 日期：2026-03-01

---

## 一、时间系统调整

### 1.1 参数变更

| 参数 | 原值 | 新值 | 影响 |
|------|------|------|------|
| 每季天数 | 28天 | 25天 | 总天数 112→100 |
| 每天现实时长 | 20分钟（1200秒） | 15分钟（900秒） | 总时长 37.3h→25h |
| 游戏时间范围 | 06:00-02:00 | 不变 | 20游戏小时/天 |
| 分钟tick | 每10分钟 | 不变 | 每天120个tick |
| 季节数 | 4季 | 不变 | 春夏秋冬 |

### 1.2 代码改动点

需修改 `TimeManager.cs`：
- `realSecondsPerGameDay`：1200f → 900f
- `daysPerSeason`：28 → 25
- `OnValidate` 中的硬编码检查（第72-78行）：28 → 25

### 1.3 新时间换算

- 1游戏天 = 15分钟现实时间
- 1季 = 25天 = 375分钟 ≈ 6.25小时
- 1年 = 100天 = 1500分钟 = 25小时
- 1现实秒 ≈ 1.33游戏分钟（原为1秒=1分钟）

### 1.4 天气日历调整（从28天压缩到25天）

**夏季枯萎天**（可逆）：第7/12/18日（原8/14/20）
**夏季雨天**：第1/4/6/9/16/23日（原1/4/6/10/18/26）
**秋季枯萎天**（不可逆）：第5/12/20日（原6/14/22）
**冬季下雪/融化**：按原比例压缩

## 二、事件密度设计框架

### 2.1 核心理念

原Deepseek-2剧本的问题：112天中大量"空白天"——玩家只重复浇水、砍树、睡觉，没有新事件推动。用户的核心诉求是**每天都有事做**，事件密度要高。

设计原则：
1. **事件驱动配方解锁**——每个新配方的获取都绑定一个事件/任务/NPC交互
2. **精力-素材-时间三重压迫**——前期一天做不完所有事，迫使玩家做优先级决策
3. **灵动引导**——不是弹窗式任务列表，而是NPC主动找你、环境暗示、对话触发
4. **解锁即教学**——获得新工具/配方的同时，有一个需要立刻使用它的场景

### 2.2 事件类型分类

| 类型 | 说明 | 触发方式 | 示例 |
|------|------|----------|------|
| 主线事件 | 推动剧情，解锁核心配方/区域 | 天数+条件 | 第3天老杰克教种田 |
| NPC任务 | NPC主动发起，引导玩家做事 | 好感度+天数 | 艾拉请你帮忙修篱笆 |
| 发现事件 | 玩家自然探索时触发 | 到达位置/使用物品 | 砍第10棵树触发记忆闪回 |
| 天气事件 | 天气驱动的特殊事件 | 天气系统 | 第一个枯萎天，马库斯教你应对 |
| 季节事件 | 季节转换触发 | 季节变化 | 夏季第一天，老汤姆提醒枯萎 |
| 节日 | 固定日期的社交活动 | 固定天数 | 春14春季集市 |
| 里程碑 | 村落复兴进度触发 | 累计完成数 | 修复5个设施→村民庆祝 |

### 2.3 每日事件密度标准

- **春季前10天**：每天至少1个主线事件 + 1个NPC交互（教学期，最高密度）
- **春季11-25天**：每2天至少1个事件（玩家已熟悉操作，开始自由探索）
- **夏季**：每3天至少1个主线事件，穿插天气事件和NPC任务
- **秋季**：主线事件集中在关键节点（老杰克去世等），日常以农田管理和地牢为主
- **冬季**：主线收束，真相揭示，事件集中在最后5天

## 三、核心资源循环与配方解锁时间线

### 3.1 资源循环闭环

```
砍树 → 木材 → 造工具/修设施
          ↓
        锯木台 → 木板 → 建造/升级

种田 → 农作物 → 食物（恢复精力）
                    ↓
                  烹饪台 → 高级食物（Buff）

挖矿 → 矿石 → 熔炉 → 金属锭 → 工具升级/武器
                                    ↓
                                  地牢探索 → 记忆碎片 → 解锁高阶配方
```

关键设计：**每个环节都消耗精力，食物是唯一的精力恢复途径（除了睡觉），所以种田从第一天就必须开始。**

### 3.2 配方解锁时间线（春季）

这是最关键的部分——春季25天内，玩家必须解锁所有基础玩法。

| 天数 | 解锁内容 | 触发方式 | 设计意图 |
|------|----------|----------|----------|
| 第1天 | 手工制作（木棍、火把） | 主线：马库斯引导 | 最基础的制作教学 |
| 第1天 | 基础锄头+水壶 | 主线：老杰克赠送 | 种田入门 |
| 第2天 | 种子（大蒜/生菜） | 老杰克教种田+赠送种子 | 让玩家第2天开始种田 |
| 第3天 | 石斧配方 | 砍树3棵后记忆闪回 | 解锁采集效率提升 |
| 第4天 | 石镐配方 | 发现矿石后NPC引导 | 解锁挖矿 |
| 第5天 | 锯木台（Lv1） | 主线：修复村口篱笆任务奖励 | 木材加工入口 |
| 第7天 | 第一批作物收获 | 自然生长（大蒜4天） | 第一次收获体验+食物来源 |
| 第8天 | 基础烹饪配方 | 收获后NPC教做菜 | 食物→精力循环打通 |
| 第10天 | 熔炉（石质Lv1） | 老乔治任务：帮铁匠铺搬矿石 | 金属加工入口 |
| 第12天 | 钓鱼竿 | 老汤姆任务：码头修复 | 新资源获取途径 |
| 第14天 | 春季集市（节日） | 固定日期 | 社交+交易+小游戏 |
| 第15天 | 铜工具配方 | 熔炉使用5次触发闪回 | 工具升级路线 |
| 第18天 | 地牢入口解锁 | 主线：地震→裂缝出现 | 探索系统入口 |
| 第20天 | 炼药台（Lv1） | 地牢第1层通关奖励 | 药水系统入口 |
| 第25天 | 季节总结+夏季预告 | 马库斯对话 | 承上启下 |

### 3.3 设计关键点

1. **第1-2天极度引导**：玩家不会迷茫，NPC手把手教操作
2. **第3-7天释放探索**：基础工具到手，玩家开始自由砍树挖矿种田
3. **第7天收获高潮**：第一批作物收获 → 做食物 → 精力循环完成 → 体验闭环
4. **第10天后加速**：工作台接连解锁，玩家感受到"越来越强"
5. **第18天地牢开启**：春季过半才开地牢，确保基础玩法已熟练

## 四、主角设定修正

### 4.1 穿越者方向（用户确认）

原Deepseek-2设定"千年前辉煌纪元的时空修补匠"过于宏大，与像素农场体量不匹配。

修正方向：
- **穿越前身份**：中世纪后期（或近现代）的工匠/魔法工匠，掌握先进的制造技术
- **穿越到**：中世纪之前的一个较为原始的时段，技术落后
- **穿越原因**：待定（可以是意外，也可以是使命），先不展开
- **失忆机制保留**：穿越过程中记忆受损，通过特定事件逐渐"回忆起"自己的制造技术
- **核心体验**：玩家是一个"会很多东西但想不起来"的人，每次记忆闪回 = 解锁配方/图纸
- **与NPC的关系**：村民觉得你"似乎天生就会做东西"，但你自己也不知道为什么

### 4.2 与玩法的绑定

- 记忆闪回 = 配方解锁（游戏机制）
- "穿越者见过更先进的东西" = 解释为什么能升级工作台
- 失忆 = 玩家前期只能用基础工具（叙事合理化）
- 主角的衣着/风格与村民不同 = 玩家角色视觉辨识度

## 六、春季逐日事件编排（前10天详细版）

> 以下是春季前10天的逐日事件设计。这是事件密度最高的教学期，每天都有明确的引导和新内容。后续天数由智能体基于此框架展开。

### 春1日 —— 觉醒

**主线事件**：
- 06:00 醒来，在一间简陋的小屋中（马库斯家），失忆状态
- 马库斯出现对话：简述情况（你昏倒在村口，我们救了你，你什么都不记得）
- 马库斯引导你出门，介绍村落基本布局（走到3个标记点）
- 走到工作台旁 → 触发第一次微弱记忆闪回（"这个东西...我好像知道怎么用"）→ 解锁手工制作

**NPC交互**：
- 路遇老杰克，简短对话："哦，你醒了？看你的手，粗糙得像个干活的人。"
- 赠送基础锄头+水壶："这些放着也没用，你拿去吧。"

**玩家可做的事**：
- 手工制作木棍（教学引导）
- 试用锄头挖地（1-2格，体验操作）
- 探索村落（熟悉地图）

**精力预算**：200精力，挖地约消耗30，砍树约消耗40，剩余130用于探索
**设计意图**：第一天以叙事和引导为主，不施加太大压力，让玩家熟悉操作

---

### 春2日 —— 第一粒种子

**主线事件**：
- 老杰克在农田区等你，主动对话
- 教学：完整的种田流程（开垦→播种→浇水）
- 赠送种子袋：大蒜×1袋（4粒）+ 生菜×1袋（3粒）
- 引导你种下去，并解释"每天要浇水"

**NPC交互**：
- 艾拉路过打招呼："杰克爷爷可喜欢教人种地了，你有福了。"（好感+5）

**玩家行为引导**：
- 种田（7格），浇水（7格）→ 精力消耗约 70
- 砍树收集木材（为明天做准备）→ 精力消耗约 60
- 剩余精力自由探索

**设计意图**：第二天开始种田，确保第6-7天能收获，形成循环闭合

---

### 春3日 —— 石器时代

**发现事件**：
- 砍第3棵树时，触发记忆闪回（画面闪白+文字："...把石头绑在木棍上...更锋利..."）
- 解锁配方：石斧、石镐

**NPC交互**：
- 马库斯找你谈话："你好像突然就会了？...也许你以前是个工匠？"（推进身世线索）

**玩家行为引导**：
- 收集石头（挖地或捡地面石块）
- 制作石斧（提升砍树效率）
- 制作石镐（为后续挖矿做准备）
- 日常浇水

**设计意图**：第三天给第一次效率提升体验，玩家感受到"变强了"

---

### 春4日 —— 矿脉

**主线事件**：
- 用石镐在矿区采集时，发现铜矿石
- 老乔治路过（他每天在铁匠铺附近徘徊但不进去）
- 简短对话："铜矿？没用的。就算有矿，我这铁匠铺三年没开火了。"（伏笔）

**玩家行为引导**：
- 挖矿（收集铜矿+石头）
- 日常浇水
- 继续砍树备木材

**设计意图**：引出老乔治角色线，埋铁匠铺重开的伏笔

---

### 春5日 —— 修篱笆

**NPC任务**：
- 马库斯请你帮忙：村口篱笆年久失修，牲畜跑出去了
- 任务需求：木材×15（如果前几天砍树够了直接交，不够就得再砍）
- 完成奖励：马库斯带你去锯木厂残骸，触发记忆闪回 → 解锁锯木台Lv1配方
- 制作锯木台需要：木材×20 + 石头×10

**玩家行为引导**：
- 交木材→完成任务→获得锯木台配方
- 收集材料制作锯木台（可能当天做不完，拖到第6天）
- 日常浇水

**设计意图**：第一个"任务→配方解锁"的完整循环。玩家体验到"帮人=获得新能力"

---

### 春6日 —— 木板

**发现事件**：
- 锯木台制作完成（如果昨天没做完今天做完）
- 使用锯木台加工木板（教学：放入木材→等待→取出木板）
- 木板是后续许多配方的基础材料

**NPC交互**：
- 老杰克来看你的锯木台："嚯，你这手艺可以啊。要不帮我做个新水桶？"
- 可选小任务：木板×4 → 水桶 → 赠送生菜种子×1袋

**设计意图**：锯木台的教学使用+第一个可选支线任务

---

### 春7日 —— 第一次收获

**主线事件**：（重要节点）
- 大蒜成熟！（第2天种，生长4天，第6天结束时成熟，第7天可收割）
- 收割大蒜 → 背包中多了食材
- 老杰克出现："不错不错！新鲜的大蒜，要不要我教你做点吃的？"

**触发配方解锁**：
- 老杰克教你最基础的烹饪（手工制作，不需要烹饪台）
- 解锁配方：烤大蒜（大蒜×2 → 烤大蒜，恢复15精力）
- 这是玩家第一次通过食物恢复精力

**设计意图**：这是游戏体验的关键闭环点——种田→收获→做食物→恢复精力→能干更多活。从第7天起，玩家的精力不再只靠睡觉恢复，游戏节奏加快。

---

### 春8日 —— 自由日

**无主线事件**（第一个"呼吸日"）

**引导方向**：
- 继续种更多作物（用收获的钱买新种子）
- 用锯木台批量加工木板
- 挖矿积累铜矿
- 与NPC自由对话提升好感

**设计意图**：连续7天高密度引导后，给玩家一天自由消化。但因为有浇水、锯木、挖矿等日常，这一天不会无聊。

---

### 春9日 —— 码头

**NPC任务**：
- 老汤姆在码头叹气，你走过去
- 对话："码头烂了三年了，没人修。你要是能帮忙...我可以教你捕鱼。"
- 任务需求：木板×10 + 石头×5
- 完成奖励：老汤姆赠送钓鱼竿 + 教学钓鱼操作

**设计意图**：解锁钓鱼 = 新的资源获取途径+新的精力管理选择（鱼可以做食物）

---

### 春10日 —— 重燃炉火

**主线事件**：（重要节点）
- 你已积累了一些铜矿石，但没有地方冶炼
- 老乔治今天站在铁匠铺门口，你可以主动搭话
- 对话："你...你有铜矿？...三年了，我一直没打开过这扇门。我儿子就是在那个矿洞里..."（情感铺垫）
- 任务：帮老乔治搬运矿石进铁匠铺（走3个搬运点）+ 提供木材×10作为燃料
