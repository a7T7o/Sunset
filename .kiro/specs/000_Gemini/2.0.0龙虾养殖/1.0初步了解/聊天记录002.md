## 四、部署教程（针对R9000P + 宿舍断网环境）

### Step 1：本地部署 OpenClaw（纯本地版，用你的2api）

OpenClaw现在已经非常成熟，提供了Windows下的**一键安装脚本**，不用再像以前那样手动配Node.js了。

**操作步骤**（复制下面命令到PowerShell，管理员身份运行）：
```bash
# 1. 一键安装OpenClaw（会自动装Node.js和依赖）
iwr -useb https://openclaw.ai/install.ps1 | iex

# 2. 安装完成后，进入配置目录
cd ~/.openclaw

# 3. 修改配置文件，对接你的2api中转站
# 编辑 openclaw.json，找到 model_providers，添加你的2api信息
# 重点：将 base_url 改成你的中转地址，api_key 改成你的中转key

# 4. 启动OpenClaw服务
openclaw start
```

启动成功后，浏览器打开 `http://localhost:18789` 就能看到控制台。

### Step 2：给 OpenClaw 安装“干活”的技能

OpenClaw的功能靠 **Skills（技能）** 扩展。针对你的需求，必须装这几个：
```bash
# 安装Unity开发套件（可以生成C#脚本模板）
openclaw skill install unity-editor

# 安装命令行工具（让它能操作你的终端）
openclaw skill install shell-executor

# 安装生图技能
openclaw skill install image-generation

# 安装网页搜索技能
openclaw skill install web-search
```

### Step 3：云端部署 MaxClaw（解决断网问题）

鉴于你宿舍晚上断网，花一杯奶茶钱租个云服务器是性价比最高的选择。**阿里云现在有一键部署OpenClaw的镜像**，5分钟搞定。

1. **访问**：[阿里云OpenClaw一键部署页面](https://www.aliyun.com/activity/ecs/clawdbot) 
2. **选择配置**：
   - 镜像：选择 **OpenClaw(Moltbot)镜像**
   - 地域：选**香港**（免备案，且联网搜索功能不受限）
   - 实例：2核4G（足够了，大概几十块一个月）
3. **点击“一键购买并部署”**。
4. **配置API-Key**：
   - 部署完成后，在控制台找到你的服务器，**放行18789端口**。
   - 执行命令，填入你的**阿里云百炼API-Key**（如果你用2api，也可以填2api地址，但云服务器直接调用百炼会更稳）。
5. **访问**：浏览器打开 `http://你的服务器IP:18789`，设置密码，开始使用。

## 六、关于“在Unity里自动种树”的具体实现

这是你需求里最硬核的部分，Gemini没给出答案，我来补上：

OpenClaw不能直接“动鼠标”去拖拽模型，但Unity是**可编程**的。正确的姿势是：

1. **让OpenClaw写一个Unity Editor脚本**：
   你告诉它：“帮我写一个Editor脚本，功能是在当前场景的选定位置，实例化一个指定的树模型预制体。”
2. **OpenClaw生成C#代码**，比如：
   ```csharp
   [MenuItem("Tools/Place Tree")]
   static void PlaceTree() {
       GameObject treePrefab = Resources.Load<GameObject>("TreePrefab");
       if (Selection.activeGameObject != null) {
           Instantiate(treePrefab, Selection.activeGameObject.transform.position, Quaternion.identity);
       }
   }
   ```
3. **你把这个代码复制到Unity里运行**。
4. **更进一步**：你可以让OpenClaw通过命令行调用Unity的**批处理模式**（Batch Mode），实现无人值守自动建场景。比如：
   `openclaw exec "在Unity项目路径`Assets/Scenes/Game.unity`的场景原点周围10米范围内，随机放置10棵不同的树模型。”`
