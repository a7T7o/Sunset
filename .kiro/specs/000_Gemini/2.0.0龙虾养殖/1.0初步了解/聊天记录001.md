### 🚀 OpenClaw 极简本地部署指南（R9000P 专属版）

既然你已经有了 Node.js 环境的基础概念，我们在你的 Windows (或 WSL 环境) 下直接开始：

**第 1 步：安装核心框架**
打开你的命令行终端（Terminal），确保你的 Node.js 版本在 22 以上，输入以下命令全局安装：
`npm install -g openclaw`

**第 2 步：配置你的 2api 中转站**
OpenClaw 启动后会在你的电脑生成一个配置文件目录。找到并编辑 `~/.openclaw/openclaw.json`（Windows 下通常在 `C:\Users\你的用户名\.openclaw\`）：

* 将 `base_url` 修改为你的 2api 中转站地址。
* 将 `api_key` 填入你的中转站密钥。
* 模型建议设置为目前最聪明的（如 `claude-3-5-sonnet` 或 `gpt-4o`）。

**第 3 步：注入“灵魂”与学习资料 (关键！)**
这是它学习你工作流的核心步骤。进入 `~/.openclaw/workspace/` 文件夹：

* **编辑 `SOUL.md`：** 在这里用大白话写下它的设定：“你是一个资深独立游戏全栈开发助手，熟练使用 Unity 和 CLI，你需要帮我寻找资源、写 C# 脚本、生成图片，并每天为我整理游戏圈新闻。”
* **喂养资料库：** 把你的游戏设计文档（GDD）、剧本草案、甚至你的代码规范直接丢进这个文件夹，它会自动读取并理解你的项目上下文。

**第 4 步：安装技能 (Skills) 与启动**

* 安装生图和搜索技能（命令行输入）：
`openclaw skill install web-search`
`openclaw skill install image-generation` (需在配置中填入对应的生图 API)
* 启动它并绑定你的常用通讯软件（例如飞书/Telegram）：
`openclaw start`
