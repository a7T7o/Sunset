4. 某线程准备做破坏性 Git 动作。

## scene-build 特别说明
- `scene-build` 当前仍在旧 worktree 里跑，但它已经被定性为“待迁移的独立项目现场”，不是 Sunset 普通线程。
- 在治理线程拿到冻结回执前，不直接硬搬。
- 正式迁移目标路径固定为：
  - `D:\Unity\Unity_learning\SceneBuild_Standalone\scene-build-5.0.0-001`
- 正式迁移方式固定为：
  - 先冻结当前一刀
  - 再执行 `git worktree move "D:\Unity\Unity_learning\Sunset_worktrees\scene-build-5.0.0-001" "D:\Unity\Unity_learning\SceneBuild_Standalone\scene-build-5.0.0-001"`
  - 再复核 `git worktree list --porcelain`
- 只有在下面条件同时满足时才允许迁移：
  1. `scene-build` 已回执当前完成面
  2. 当前最小施工切口已 checkpoint，或至少能明确冻结点
  3. 当前 worktree 没有正在写的 Unity / MCP live 会话
  4. 目标路径不存在冲突目录
