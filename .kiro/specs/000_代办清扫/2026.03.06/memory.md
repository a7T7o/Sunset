# 000_代办清扫 - 开发记忆

## 模块概述
本工作区用于清扫 `.kiro/specs/000_代办/` 的历史代办，结合当前代码、场景、资源与 steering 文档，判断哪些代办可执行、阻塞、过时或已被后续实现覆盖，并输出按系统分块的审查报告。
