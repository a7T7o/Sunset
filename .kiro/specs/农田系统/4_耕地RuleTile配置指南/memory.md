# 4_耕地RuleTile配置指南 - 子工作区记忆

## 概述
为用户提供详细的 Rule Tile 配置指南，包括 Sprite 准备、规则配置、场景 Tilemap 创建等。

## 当前状态
- **完成度**: 100%
- **最后更新**: 2026-01-25

## 核心内容

### 资源清单
- 耕地 Sprite：上、下、左、右、中心（需要手动合成完整边界）
- 水渍 Sprite：3 种样式

### 配置步骤
1. 创建 Rule Tile 资产
2. 配置 6 条规则（孤立、上边界、下边界、左边界、右边界、中心）
3. 创建水渍 Tile（3 个）
4. 在场景中创建 Tilemap
5. 测试 Rule Tile
6. 配置 FarmingManager
