# 耕地判定问题分析报告

**创建日期**: 2026-03-02
**问题描述**: 耕地判定存在大量问题，需要剖析代码并给出解决方案
**用户需求**: 1+8范围内（中心块全部+周围8格半格范围，即1.5单位正方体）不可有任何碰撞体，需要标签+碰撞体双层判定

---

## 1. 问题概述

### 1.1 用户描述的预期行为

用户期望的耕地判定规则：
1. **范围定义**: 1+8范围 = 1个中心块的全部 + 周围8格的半格范围
2. **等价描述**: 以中心块中心为原点，1.5单位长度的正方体内不可有任何碰撞体
3. **判定机制**: 标签批量选择 + 碰撞体双层判定（类似导航系统）
4. **特殊物体**: 稻草人、洒水器等后续物品不影响耕地（需要后续补充）
5. **楼层隔离**: 不在同一层的物体不应影响耕地判定

### 1.2 当前实现的实际行为

根据代码分析，当前实现：
- **检测范围**: `Physics2D.OverlapBoxAll(cellCenter, new Vector2(0.9f, 0.9f), 0f)`
- **检测方法**: 碰撞体检测 + 标签过滤 + 特殊对象检测（树苗、箱子）
- **标签列表**: `FarmingObstacleTags`（静态数组，不包含 Player）
- **楼层检测**: 无楼层隔离机制

## 4. 根因分析

### 4.1 为什么检测范围是 0.9f？

**推测**:
- 原设计可能只想检测中心格子内的障碍物
- `0.9f` 是为了避免边界误判（1.0f 可能会检测到相邻格子的碰撞体）
- **但这与用户需求（1.5单位正方体）严重不符**

### 4.2 为什么树苗/箱子只检测中心格子？

**推测**:
- `HasTreeAtPositionStatic()` 和 `HasChestAtPositionStatic()` 使用格子索引匹配
- 只比较中心格子的索引，未遍历周围8格
- **这是明显的逻辑缺陷**

### 4.3 为什么没有楼层隔离？

**推测**:
- `Physics2D.OverlapBoxAll()` 是2D物理检测，不考虑Z轴
- Unity 2D 物理系统本身不支持楼层概念
- **需要手动过滤碰撞体的 Sorting Layer 或自定义楼层标识**

## 5. 解决方案

### 5.1 方案A：扩大检测范围（推荐）

#### 5.1.1 修改 `HasFarmingObstacle()`

```csharp
/// <summary>
/// 检测1+8范围内是否有障碍物
/// 范围：以中心格子为原点，1.5单位正方体（覆盖中心块+周围8格半格）
/// </summary>
public static bool HasFarmingObstacle(Vector3 cellCenter, int targetLayerIndex = 0)
{
    // 1. 碰撞体检测（扩大到1.5单位）
    Vector2 boxSize = new Vector2(1.5f, 1.5f);  // ✅ 修复P1
    Collider2D[] hits = Physics2D.OverlapBoxAll(cellCenter, boxSize, 0f);

    foreach (var hit in hits)
    {
        // ✅ 修复P5: 楼层隔离
        if (!IsSameLayer(hit.transform, targetLayerIndex))
            continue;

        // ✅ 修复D1: 白名单机制
        if (IsInWhitelist(hit.transform))
            continue;

        if (HasAnyTagStatic(hit.transform, FarmingObstacleTags))
        {
            return true;
        }
    }

    // 2. 检测无碰撞体的树苗（1+8范围）
    if (HasTreeInArea(cellCenter, 1.5f, targetLayerIndex))  // ✅ 修复P3
        return true;

    // 3. 检测无碰撞体的箱子（1+8范围）
    if (HasChestInArea(cellCenter, 1.5f, targetLayerIndex))  // ✅ 修复P4
        return true;

    return false;
}

/// <summary>
/// 检测区域内是否有树木（遍历周围格子）
/// </summary>
private static bool HasTreeInArea(Vector3 center, float radius, int targetLayerIndex)
{
    Vector2Int centerCell = PlacementGridCalculator.GetCellIndex(center);

    // 遍历3x3格子（中心+周围8格）
    for (int dx = -1; dx <= 1; dx++)
    {
        for (int dy = -1; dy <= 1; dy++)
        {
            Vector2Int checkCell = new Vector2Int(centerCell.x + dx, centerCell.y + dy);
            Vector3 checkCellCenter = PlacementGridCalculator.GetCellCenter(checkCell);

            // 检查该格子是否有树木
            var allTrees = Object.FindObjectsByType<TreeController>(FindObjectsSortMode.None);
            foreach (var tree in allTrees)
            {
                // 楼层隔离
                if (!IsSameLayer(tree.transform, targetLayerIndex))
                    continue;

                Vector3 treeRootPos = tree.transform.parent != null
                    ? tree.transform.parent.position
                    : tree.transform.position;

                Vector2Int treeCellIndex = PlacementGridCalculator.GetCellIndex(treeRootPos);

                if (checkCell == treeCellIndex)
                    return true;
            }
        }
    }

    return false;
}

/// <summary>
/// 检测区域内是否有箱子（遍历周围格子）
/// </summary>
private static bool HasChestInArea(Vector3 center, float radius, int targetLayerIndex)
{
    Vector2Int centerCell = PlacementGridCalculator.GetCellIndex(center);

    // 遍历3x3格子（中心+周围8格）
    for (int dx = -1; dx <= 1; dx++)
    {
        for (int dy = -1; dy <= 1; dy++)
        {
            Vector2Int checkCell = new Vector2Int(centerCell.x + dx, centerCell.y + dy);

            var allChests = Object.FindObjectsByType<ChestController>(FindObjectsSortMode.None);
            foreach (var chest in allChests)
            {
                // 楼层隔离
                if (!IsSameLayer(chest.transform, targetLayerIndex))
                    continue;

                var collider = chest.GetComponentInChildren<Collider2D>();
                Vector3 chestPos = collider != null ? collider.bounds.center : chest.transform.position;
                Vector2Int chestCellIndex = PlacementGridCalculator.GetCellIndex(chestPos);

                if (checkCell == chestCellIndex)
                    return true;
            }
        }
    }

    return false;
}

/// <summary>
/// 检查 Transform 是否在目标楼层
/// </summary>
private static bool IsSameLayer(Transform t, int targetLayerIndex)
{
    // 方法1: 通过 Sorting Layer 判断（推荐）
    var renderer = t.GetComponentInChildren<SpriteRenderer>();
    if (renderer != null)
    {
        string layerName = SortingLayer.IDToName(renderer.sortingLayerID);
        // 假设楼层命名为 "Layer 1", "Layer 2", "Layer 3"
        int layerIndex = layerName == "Layer 1" ? 0 :
                         layerName == "Layer 2" ? 1 :
                         layerName == "Layer 3" ? 2 : 0;
        return layerIndex == targetLayerIndex;
    }

    // 方法2: 通过父级 Transform 名称判断（备用）
    Transform current = t;
    while (current != null)
    {
        if (current.name.Contains("LAYER 1") && targetLayerIndex == 0) return true;
        if (current.name.Contains("LAYER 2") && targetLayerIndex == 1) return true;
        if (current.name.Contains("LAYER 3") && targetLayerIndex == 2) return true;
        current = current.parent;
    }

    return true; // 默认认为在同一层（兼容旧代码）
}

/// <summary>
```
