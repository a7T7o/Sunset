# V 键统一放置模式方案分析报告

## 一、需求概述

### 1.1 核心需求

用户需求：新增 V 键作为"放置模式"的 toggle 开关，统一控制耕地工具（锄头、水壶）和放置物品（树苗、箱子、种子）的预览和交互。

默认状态（未按 V）：
- 锄头：只能除草，不能耕地
- 水壶：不能浇水
- 放置物品：不显示预览，不能放置

放置模式（按 V 后）：
- 锄头：可以耕地，显示耕地预览
- 水壶：可以浇水，显示浇水预览
- 放置物品：显示放置预览，可以放置

### 1.2 交互设计

V 键行为：
- 单击 V 键：toggle 进入/退出放置模式
- 后续会引入 UI 显示当前是否处于放置模式

工具切换行为：
- 切换工具不影响放置模式状态
- 例如：在放置模式下从锄头切换到树苗，仍然保持放置模式

其他按键行为：
- ESC/WASD：保持现有逻辑（清除队列预览、退出放置模式）
- 与 V 键独立，不冲突

## 三、方案设计

### 3.1 架构设计

#### 方案 A：新增 PlacementModeManager（推荐）

创建独立的 `PlacementModeManager` 单例，管理统一放置模式状态。

优点：
- 职责清晰，单一责任原则
- 易于扩展和维护
- 不影响现有 PlacementManager 的逻辑

缺点：
- 需要新建脚本

#### 方案 B：扩展现有 PlacementManager

在现有 `PlacementManager` 中添加统一放置模式状态。

优点：
- 不需要新建脚本
- 放置相关逻辑集中

缺点：
- PlacementManager 职责变重
- 可能与现有 Q 键逻辑冲突

推荐：方案 A

### 3.2 状态管理

```csharp
public class PlacementModeManager : MonoBehaviour
{
    private static PlacementModeManager _instance;
    public static PlacementModeManager Instance => _instance;
    
    // 统一放置模式状态
    private bool _isInUnifiedPlacementMode = false;
    public bool IsInUnifiedPlacementMode => _isInUnifiedPlacementMode;
    
    // 事件：放置模式切换
    public event System.Action<bool> OnPlacementModeChanged;
    
    // Toggle 放置模式
    public void TogglePlacementMode()
    {
        _isInUnifiedPlacementMode = !_isInUnifiedPlacementMode;
        OnPlacementModeChanged?.Invoke(_isInUnifiedPlacementMode);
    }
    
    // 强制退出放置模式（ESC/WASD 触发）
    public void ExitPlacementMode()
    {
        if (_isInUnifiedPlacementMode)
        {
            _isInUnifiedPlacementMode = false;
            OnPlacementModeChanged?.Invoke(false);
        }
    }
}
```

## 七、总结

### 7.1 方案优点

1. 交互隔离：彻底隔离耕地工具和放置物品的交互，避免误触
2. 统一体验：所有放置类操作都通过 V 键控制，用户体验一致
3. 扩展性强：后续可以轻松添加新的放置类工具或物品
4. 状态清晰：通过 UI 显示放置模式状态，用户不会困惑

### 7.2 方案缺点

1. 学习成本：用户需要学习新的 V 键操作
2. 操作步骤增加：需要先按 V 键才能进行放置操作
3. 可能的遗忘：用户可能忘记按 V 键，导致操作无效

### 7.3 改进建议

1. 新手引导：在游戏开始时提示用户 V 键的功能
2. 视觉反馈：在未进入放置模式时，鼠标悬停在工具上显示提示"按 V 键进入放置模式"
3. 音效反馈：进入/退出放置模式时播放音效
4. 快捷切换：考虑添加"按住 V 键"的临时放置模式（松开自动退出）

```csharp
public class PlacementModeUI : MonoBehaviour
{
    [SerializeField] private GameObject placementModeIndicator;
    [SerializeField] private Text placementModeText;
    
    void Start()
    {
        // 订阅放置模式变化事件

    }
    
    private void UpdateUI(bool isInPlacementMode)
    {
        if (placementModeIndicator != null)
        {
            placementModeIndicator.SetActive(isInPlacementMode);
        }
        
        if (placementModeText != null)
        {
            placementModeText.text = isInPlacementMode ? "放置模式" : "";
        }
    }
}
```
